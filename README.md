# 🦊 猛兽派对钓鱼辅助工具（Party Animals Fishing Helper）

> ⚠️ **本项目仅供个人学习与研究使用，禁止用于商业用途或干扰他人游戏体验。请遵守游戏规则和社区准则。**

这是一个基于 Python 的自动化钓鱼辅助脚本，专为 Steam 游戏 **《猛兽派对》（Party Animals）** 设计。它通过实时截图分析游戏画面，自动完成抛竿、咬钩检测、收杆、鱼桶满提示等操作，并支持五角星品质识别与音效反馈。

---

## 🔧 功能特点

- ✅ **智能咬钩检测**：通过数字区域变化判断是否上钩
- ✅ **鱼桶满自动检测**：监控 F 按钮区域变化，防止鱼桶溢出
- ✅ **五角星品质识别**：自动识别“标准/非凡/稀有/史诗/传奇”并统计数量
- ✅ **音效反馈**：不同品质触发不同语音（需提供对应 `.wav` 文件）
- ✅ **悬浮日志窗口**：半透明 GUI 显示状态、统计和操作提示
- ✅ **热键控制**：
  - `F1`：启动 / 暂停钓鱼
  - `F2`：退出程序
- ✅ **高兼容性**：支持 PyInstaller 打包，适配开发与发布模式

---

## ⚙️ 使用要求

### 系统环境
- Windows 10 / 11（依赖 `win32gui`, `winsound`）
- Python 3.8+

### 依赖库
```bash
pip install opencv-python numpy pyautogui mss pygetwindow keyboard Pillow pywin32
```

### 资源文件（需放在项目根目录）
- `star_template.png`：五角星识别模板
- 音频文件（可选）：
  - `bucket_full.wav`：鱼桶满提示音
  - `ouhuang.wav`：传奇鱼提示音
  - `huiben.wav`：稀有/史诗提示音
  - `dawo.wav`：普通鱼提示音

> 💡 音频文件缺失时会跳过播放，不影响核心功能。

---

## 🚀 快速开始

1. **启动《猛兽派对》**，进入钓鱼小游戏界面，面朝小河。
2. **运行主程序**：
   ```bash
   python main.py
   ```
3. 程序会自动：
   - 定位游戏窗口
   - 显示悬浮控制窗
   - 提示“按 F1 开始钓鱼”
4. 按下 `F1` 即可启动自动钓鱼流程。
