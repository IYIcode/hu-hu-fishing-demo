

# 猛兽派对钓鱼辅助

> ⚠️ **仅用于学习 Python 自动化与图像识别技术，请勿用于违反游戏规则。**

这是一个基于 OpenCV 和 PyAutoGUI 的自动化钓鱼工具，适用于《Party Animals》（猛兽派对）。它能自动识别钓鱼时机并触发操作，支持鱼桶满了语音提示。

---

## 📦 功能特点

- 🎯 实时检测钓鱼进度条（基于模板匹配）
- 🔊 桶满语音反馈：鱼品质音效
- 🖱️ 热键控制：`F1` 启动，`F2` 停止
- 🧪 轻量无后台，关闭即停

---

## 🛠️ 使用方式

### 方式一：直接运行（需 Python 环境）
1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 运行主程序：
   ```bash
   python yt.py
   ```

### 方式二：使用预编译版（推荐普通用户）
👉 前往 [Releases](https://github.com/IYIcode/hu-hu-fishing-demo/releases) 下载 `.exe` 文件，无需安装 Python。

---
▶️先打开游戏再运行脚本或者程序
🎮选择1024×768分辨率（基于该分辨率开发的，若需要支持更多分辨率自行开发），拿出鱼竿，准备好钓饵，面朝小河或钓洞，最好坐在凳子上
📖按F1开始钓鱼（绿色亮），中途不钓再按F1（红灯亮）此时会等到完成本次钓鱼再暂停，准备好钓鱼后再按F1开始，全程可按F2退出程序【注意热键冲突】
📌鱼桶满了会有声音提示

---

## 🧱 构建可执行文件

运行以下命令生成独立 `.exe`：
```bat
auto-exe.bat
```
生成的程序位于 `dist/` 目录下。

> 💡 你也可以手动打包：
> ```bat
> pyinstaller --onefile --icon=resource/vicksy.ico yt.py
> ```
